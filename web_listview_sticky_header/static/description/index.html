OVERVIEW

This application ensures that an Odoo user never gets lost in a long list of records again as it sticks the header of Tree View and enhances user experience by mega folds.

FEATURES

  • This module enhance the user experience by sticking the header of tree View so that user will never get lost in a long list of records.
  • Horizontal scrolling of table header now possible on horizontal scroll. (Update)
